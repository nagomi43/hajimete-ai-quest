# はじめてAIクエスト

AI初心者が、ガチャで出たミッションをクリアしながら「AIへのお願い文」を作るミニゲームです。

OpenAI APIは使っていません。作ったお願い文をコピーして、ChatGPTなどに貼って遊べます。

## できること

- ガチャでミッションを表示
- 質問に答えてAIへのお願い文を作成
- お願い文をコピー
- 「できた！」でEXPを獲得
- レベル、称号、クリア履歴をブラウザに保存
- スマホ対応
- GitHub Pagesで公開しやすい相対パス設定

## ローカルで動かす

```bash
npm.cmd install
npm.cmd run dev
```

表示されたURLをブラウザで開きます。

## 公開用ファイルを作る

```bash
npm.cmd run build
```

`dist` フォルダに公開用ファイルが作られます。

## GitHub Pagesで動かす方法

1. GitHubで新しいリポジトリを作ります。
2. このフォルダの中身をリポジトリにアップロードします。
3. リポジトリの Settings → Pages で Source を `GitHub Actions` にします。
4. `main` ブランチへ反映すると、自動でビルドと公開が走ります。
5. Vite設定で `base: './'` にしているので、リポジトリ名が変わっても動きやすい構成です。
